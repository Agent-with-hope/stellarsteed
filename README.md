这是一个专门为 2026 马年设计的沉浸式数字贺卡，旨在为朋友提供一种新颖的节日祝福体验。

视觉呈现：

利用 Three.js 开发的高精度 3D 纸灯笼模型，具备模拟真实物理感的“呼吸灯”光效。

采用 Scrollytelling（滚动叙事） 技术，随着用户滑动页面触发不同的动画阶段。

AI 交互模块：

赛博春联生成器：用户输入关键词（如“搞钱”），系统实时生成符合 Z 世代语境的幽默对联。

赛博判官聊天室：内置一个性格鲜明的 AI 嘴替，支持幽默互怼、默契度测试以及盲盒锦囊抽取。

数据可视化：

通过 Chart.js 绘制“精神状态报告”和“搞钱运势大盘”，将感性的友谊通过理性的雷达图和折线图具象化。
